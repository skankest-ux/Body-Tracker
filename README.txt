Body Tracker v7.0.0

Drop these files into the root of your GitHub Pages repository and commit.
Before updating, export a JSON backup from the app.
After GitHub Pages redeploys, open the installed web app and use Settings > Update app.

v7 highlights:
- Flexibility naming instead of Mobility in UI
- Benchmarks page replaces Trends/Measurements language
- Estimated Body Fat is calculated, not manually entered
- Weight and waist are benchmarks
- Profile settings are stable info: height, birthdate, sex/gender baseline
- Weekly pace uses ahead/behind number instead of expected-by-today marker
- Profile bars collapse by default
- Benchmark test instructions added
- Single Submit Benchmarks button overwrites the selected day
- Activity edit workflow restored
